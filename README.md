# WAIC 2026 · 家庭场景 Demo (sudo)

围绕机器人「pick & place」核心能力的家庭场景交互原型。三个空间、五个任务，用户在 iPad 上选择意图，机器人按规则完成；部分任务支持与机器人 PK。

**在线预览**: https://sudoai-labs.github.io/waic-playground/

## 场景
| 空间 | 任务 | 核心能力 | 人机挑战 |
|---|---|---|---|
| 书房 | 整理书桌 | 朝向控制（标签朝外） | — |
| 餐厅 | 餐具摆台 | 规范摆放 + 多位对称 | — |
| 厨房 | 垃圾分类 | 规则推理 + 分拣 | ✅ |
| 儿童房 | 玩具收纳 | 类别分拣 | ✅ |
| 儿童房 | 乐高分拣 | 颜色分拣（自重置） | — |

## 说明
- 纯静态 HTML/CSS/JS，无构建步骤。入口 `index.html`。
- 按 iPad 横屏（1180×820）设计，建议桌面浏览器查看。
- 本地预览：`python3 -m http.server` 后打开 `index.html`。
